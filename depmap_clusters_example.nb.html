Clustering Anchor Hits

Setup

Load our libraries first

library(tidygraph)

package tidygraph was built under R version 3.5.2
Attaching package: tidygraph

The following object is masked from package:stats:

    filter

Load data and set parameters

The gene hits data must have a Gene ID column with NCBI gene IDs. All other columns are treated as metainformation.

Gene effect scores can be downloaded from the depmap.

Create Network

Now we want to extract the relevant columns and create our network.
